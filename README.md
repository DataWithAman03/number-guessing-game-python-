# Number Guessing Game (Python)

A beginner-friendly Python number guessing game where the computer generates a random number between 1 and 100, and the user tries to guess it using hints like “Higher” or “Lower.”

This project includes:

* while loop and for loop versions
* conditional statements
* random module usage
* guess counting
* attempt limits
* basic game logic

Files:

* `while_loop_version.py`
* `for_loop_version.py`

This project was created for practicing Python fundamentals and logic building.
